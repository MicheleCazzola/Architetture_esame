The LandTiger LPC17XX development board has onboard support for 10/100 Mbit/s Ethernet communication. The LPC1768 chip supports the RMII interface and links to the DP83848 (U5) Ethernet PHY chip. This device interfaces out via the RJ45 connector (CN5) which has internal magnetics and network filters. The RJ45 connector has integrated LEDs to indicate link connectivity and collisions. A separate LED indicates 10/100 Mbit/s linkspeed (LD2).
