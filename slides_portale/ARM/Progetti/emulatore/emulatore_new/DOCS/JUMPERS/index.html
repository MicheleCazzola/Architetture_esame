Jumper Description
JP1 JP1 is used to connect onboard backup battery to the RTC. Battery backup: Jumper inserted No battery backup: Jumper removed
JP2 JP2 is used to enable the external speaker. Speaker connected to P0.26: Jumper short No Speaker output: Jumper removed
JP3 JP3 is used to select the external 5V power supply or USB powersupply. External 5V power supply: Jumper short1-2 USB-powered: Jumper short 2-3
JP4 JTAG_SEL Insert Jumper to enable external JTAG device. Remove Jumper to enable onboard JLINK emulator
JP5 INT0 (P2.10) connected to Key2 Insert Jumper to enable manual entering ISP mode
JP6 Enable ISP Select. Enable ISP Select: JP6 inserted Disable ISP Select: JP6 removed
Insert Jumper to enable external entering ISP mode
JP7 Enable ISP Reset. Enable ISP Reset: JP7 inserted Disable ISP Reset: JP7 removed
Reset activated by DTR Insert Jumper to enable external Reset
JP8 Enable databus LEDs. LEDs enabled: JP8 inserted LEDs disabled: JP8 removed
JP9 Select USB-D +. USB device: JP9 set to 1-2
USB host: JP9 set to 3-4
JP10 Select USB-D + signal pull-up resistor mode. LED LD14 is lit when the pull-up is activated. USB pull-up control by P2.9: JP10 set to 1-2 (P2.9 control signal is active low). USB permanent pull-up: JP10 set to 2-3
JP11 Select USB-D -. USB device: JP11 set to 1-2
USB host: JP11 set to 3-4
JP12 JP12 is used to enable the VR1 connection to ADC input (P1.31 AD0.5). VR1 Connected to AD0.5: Jumper short No ADC input: Jumper removed
